About this utility

copyright 2016 Petras Jokubauskas <klavishas@gmail.com>

Cam-overlap-manager is utility to read, generate, merge, and re-use the overlap files generated by Cameca™ Peaksight™ software.

Cam-overlap-manager is developed independently from Cameca or Ametek inc. Development is based on the RE of binary data formats which is the Fair Use.

Full copy of source code can be obtained at http://www.github.com/sem-geologist/cam-overlap-manager.

The utility is written using python 3 with PyQt5 library.
